Help Me Please

1. Save/Load

Save adalah tombol yang berfungsi melakukan penyimpanan hollow objek

Load adalah tombol yang berfungsi memuat hollow objek dari komputer

2. Rotation

Rotation adalah fungsi untuk membuat object hollow 3D berotasi sesuai dengan sumbunya

  • Rotate X : object hollow berotasi pada sumbu X
  • Rotate Y : object hollow berotasi pada sumbu Y
  • Rotate Z : object hollow berotasi pada sumbu Z

3. Translasi

Translasi adalah fungsi untuk membuat object hollow 3D melakukan translasi sesuai dengan sumbunya

  • Translasi X : object hollow bertranslasi pada sumbu X
  • Translasi Y : object hollow bertranslasi pada sumbu Y
  • Translasi Z : object hollow bertranslasi pada sumbu Z

4. Scale

Scaling adalah fungsi untuk membuat object hollow 3D melakukan scale atau perubahan ukuran

5. Camera Control

Camera control berfungsi untuk mengatur kamera view dan menggerakan kamera mengitari model

Cara menggunakannya dengan menggeserkan slider

6. Reset

Button yang berfungsi untuk mengembalikan view ke settingan default/semula

7. Menu Help Me

Menu help berfungsi untuk melihat penjelasan mengenai fungsi - fungsi yang dapat digunakan.

Cara menggunakannya dengan menekan tombol Help Me Please dan akan diarahkan ke page baru yang berisi help menu

8. Projection View

Ada 3 buah projection view yang dapat digunakan

  • Orthographic View
  • Oblique View
  • Perspective View

Cara menggunakannya adalah dengan menekan button dari view yang diinginkan

9 . Shading

Menu yang berfungsi untuk mengaktifkan atau menonaktifkan shading.
