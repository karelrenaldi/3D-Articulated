Help Page

1. Animation Control

Dengan menekan tombol ini maka model objek articulated akan dimainkan animasinya. Pastikan sudah memuat/load model yang ingin dijalankan animasinya.

2. Auto Rotation Control

Fitur ini berfungsi untuk membuat model objek articulated berputar secara otomatis.

  • Rotate Off : object tidak berotasi otomatis
  • Rotate X : object berotasi otomatis pada flag sumbu X
  • Rotate Y : object berotasi otomatis pada flag sumbu Y
  • Rotate Z : object berotasi otomatis pada flag sumbu Z

3. Shading Mode

Fitur untuk memilih mode tampilan di canvas.

  • Default View : object articulated model ditampilkan secara default (mode shading off)
  • Shading Mode : mengaktifkan mode shading dan memperbarui object articulated model yang ditampilkan

4. Projection Mode

Fitur untuk memilih jenis projection view di canvas.

  • Orthographic View
  • Oblique View
  • Perspective View

5. Texture Mapping Mode

Fitur untuk mengubah tekstur dari model yang ditampilkan di canvas.

  • Image Mapping : tekstur object diambil dari gambar bertekstur kayu
  • Environment Mapping : tekstur diambil dari ruang cube map
  • Bump Mapping : diberikan tekstur bump atau benjolan pada object sehingga tidak smooth dan terlihat lebih realistis

6. Model View Control

Fitur untuk mengatur world view dengan cara merubah posisi, sudut rotasi, dan scaling dari model view matriks.

  • Slider Translasi : berisi 3 slider (sumbu X, Y, dan Z) untuk memasukkan nilai perubahan posisi dan melakukan translasi pada object
  • Slider Rotasi : berisi 3 slider (sumbu X, Y, dan Z) untuk memasukkan nilai perubahan sudut dan melakukan rotasi pada object
  • Slider Scale : berisi 3 slider (sumbu X, Y, dan Z) untuk memasukkan nilai perubahan scale dan melakukan scaling pada object

7. Model Direct Control

Terdiri dari slider-slider yang akan menggerakkan bagian sendi-sendi model.

8. Camera Setting

Fitur untuk merubah pengaturan pada camera view

  • Slider Translasi : berisi 3 slider (sumbu X, Y, dan Z) untuk memasukkan nilai perubahan posisi dan mengatur titik pusat yang diamati
  • Slider Rotasi : berisi 3 slider (sumbu X, Y, dan Z) untuk memasukkan nilai perubahan sudut dan mengatur upward direction
  • Slider Scale : berisi 3 slider (sumbu X, Y, dan Z) untuk memasukkan nilai perubahan scale dan mengatur posisi mata viewer

9. Save/Load

'Save Model' adalah tombol yang berfungsi melakukan penyimpanan articulated model

'Choose File' adalah tombol yang berfungsi memuat file articulated model dari komputer. Dengan menekan tombol akan muncul pop up untuk memilih file dari penyimpanan local.

10. Menu Help

Menu help berfungsi untuk melihat penjelasan mengenai fungsi - fungsi yang dapat digunakan.

Cara menggunakannya dengan menekan tombol Help Me dan akan diarahkan ke page baru yang berisikan help menu dan penjelasan fitur.

11. Reset

Button yang berfungsi untuk mengembalikan view ke settingan default/semula.
